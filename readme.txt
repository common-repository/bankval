BankVal is the easiest way to validate and store bank account information

== Description ==

[BankVal](https://www.unifiedsoftware.co.uk) is Unified Software’s award winning [sort code & account number validation](https://www.unifiedsoftware.co.uk/solutions/sort-code-and-account-number-validation/) software. It’s an essential cloud-based solution that allows companies to instantly check the validity of bank details entered by customers. We developed BankVal to help your company save time and money by making sure your payments and Direct Debits work first time.

BankVal is a web service. That means the [bank details validation software](https://www.unifiedsoftware.co.uk/solutions/sort-code-and-account-number-validation/), the dynamic validation rules and reference data, are all located on the BankVal servers.

This plugin enables you to interact with BankVal through an easy to use form optimised for all screen sizes that you can place anywhere on your website via a shortcode. Visitors to your website can then use the form to input sort codes and account numbers which will be automatically validated against BankVal and stored in a database table of your choice for future reference.

== Installation ==

1. Upload `bankval` directory to the `wp-content/plugins/` directory
2. Activate plugin through the 'Plugins' page in WordPress 
3. Set up your credentials and configure settings through the top-level 'Unified Software' admin page

== Frequently Asked Questions ==

= What is the BankVal form shortcode? =
`[bankval_form]`

= How do I get my own BankVal credentials? =
[Contact our sales team](https://www.unifiedsoftware.co.uk/contact/)

= How are my BankVal credentials handled? =
Your credentials are encrypted using security keys in your Wordpress installation and stored locally on your server. They are decrypted only when shown in the BankVal settings page and when sent over to the BankVal servers over a secure 256 bit SSL/TLS connection.
